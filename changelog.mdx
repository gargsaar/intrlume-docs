## Recent Updates

Stay informed about the latest enhancements to Intrlume. You can build, test, and ship voice AI agents faster with these improvements.

<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>
## New Features

- Added 15 new agent templates including delivery rescheduling, policy renewals, and course admissions.
- Expanded language support to 12+ Indian languages with improved noisy environment handling.

## Improvements

- Enhanced the live playground for instant agent testing without signup.
- Updated console interface with better real-time evaluation metrics display.

</Update>

<Update label="2024-09-20" description="v1.2.0" tags={["feature", "improvement"]}>
## New Features

- Introduced customizable evaluation rubrics with metrics for faithfulness, hallucination, and topic adherence.
- Added support for concurrent calls up to 100+ in production environments.

## Bug Fixes

- Fixed routing issues in the CREATE AGENT workflow for appointment booking agents.
- Resolved evaluation score calculation delays in the console dashboard.

</Update>

<Update label="2024-08-10" description="v1.1.0" tags={["improvement"]}>
## Improvements

- Optimized playground performance for sub-minute agent deployment.
- Enhanced API endpoints for agent configuration with better error handling.

## Bug Fixes

- Corrected phone number formatting for Indian numbers in the console.

</Update>
