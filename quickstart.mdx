## Prerequisites

<Callout kind="info">
  You need an Intrlume account. Sign up at the console to access the live playground. No credit card is required for the initial test.
</Callout>

## Access the Playground

Open the Intrlume console and navigate to the live playground. This environment lets you describe, build, and test agents without leaving the page.

<Steps>
  <Step title="Describe your agent" icon="edit-3">
    Type a natural language description in the console. For example:

    ```text
    Create an agent that books solar demo appointments, confirms time slots, and handles rescheduling requests in Hindi and English.
    ```
  </Step>
  <Step title="Generate the agent" icon="zap">
    Click **Create Agent**. The system builds the agent in seconds and assigns a phone number for testing.
  </Step>
</Steps>

## Test the Agent with Live Calls

Once generated, your agent appears in the console with status **Live**.

<Tabs>
  <Tab title="Voice Test" icon="phone">
    Dial the provided number from your phone. Speak naturally—the agent handles interruptions, confirms details, and books appointments.
  </Tab>
  <Tab title="Chat Test" icon="message-circle">
    Use the in-console chat simulator to run text-based conversations and verify responses before going live.
  </Tab>
</Tabs>

## Review Evaluation Scores

After each test call, the console automatically runs evaluations.

```json
{
  "agent_id": "ag_8c12f4",
  "calls_last_7d": 10,
  "evaluation_score": 86,
  "metrics": {
    "faithfulness": 92,
    "hallucination": 0,
    "topic_adherence": 97
  }
}
```

Review scores for faithfulness, hallucination, and topic adherence. If any metric falls below your threshold, refine the agent description and regenerate.
