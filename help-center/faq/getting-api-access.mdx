## What it's for

An **API key** gives a server-to-server integration — for example, a CRM connector — secure access to your Intrlume organization on your behalf.

## Enable API access

Server-to-server API access is **off by default** for each organization. To turn it on, email **[admin@intrlume.tech](mailto:admin@intrlume.tech)**.

## Create a key

Once it's enabled, an **admin** creates the key:

<Steps>
  <Step title="Sign in" icon="log-in">
    Go to [app.intrlume.tech](https://app.intrlume.tech) and sign in.
  </Step>
  <Step title="Open API keys" icon="key">
    Go to **Settings → API keys** (admins only).
  </Step>
  <Step title="Generate" icon="sparkles">
    Click **Generate** to create a secret **`sk_live_`** key.
  </Step>
  <Step title="Copy and store it" icon="lock">
    Copy the key and store it somewhere secure — treat it like a password.
  </Step>
</Steps>

<Callout kind="warning">
  Keep the key secret. **Regenerating** a key immediately breaks any integration still using the old one.
</Callout>

For where API keys fit alongside the rest of your account, see [Settings](/guides/settings).
