---
title: Voice Agent Quickstart
description: Deploy your first voice AI agent with Intrlume.
---

## Steps

<Steps>
  <Step title="Create a new agent">
    ````
    Click **New Agent** in the Intrlume dashboard.
    ````
  </Step>
  <Step title="Configure voice & behavior">
    ````
    Set voice model, language, and initial prompt as needed.
    ````
  </Step>
  <Step title="Test your agent">
    ````
    Use dashboard's test panel to interact via microphone or text.
    ````
  </Step>
  <Step title="Deploy">
    ````
    Click **Deploy**. Use API endpoint for production apps.
    ````
  </Step>
</Steps>
